# خطة تطوير صفحة الأعمال (Portfolio)

## الهدف
رفع مستوى صفحة `/portfolio` لتعرض كل مشروع بتفاصيل احترافية كاملة: معرض صور، مواصفات تقنية، معدات، محتويات الحوض، نطاق سعري، وزر طلب عرض سعر مماثل عبر واتساب.

---

## 1. هيكل بيانات المشروع الجديد

سيتم إنشاء ملف `src/data/projects.ts` يحوي type واحد موحّد:

```ts
type Project = {
  id, slug, title, catLabel, cat,
  location, year, duration,
  images: string[],          // معرض 2-5 صور
  cover: string,             // للبطاقة في الشبكة
  description: string,
  specs: {
    dimensions, volumeLiters, glassType, systemType
  },
  equipment: {
    filter, lighting, heatingCooling?, waveMakers?, co2?
  },
  contents: {
    fish: string[],          // قائمة أنواع
    plantsOrCorals?: string[],
    decor?: string
  },
  priceRange: { min: number; max: number; currency: "SAR" }
};
```

## 2. بيانات المشاريع (مختلطة)

- **3 مشاريع كاملة كنماذج** (فيلا الرياض، مطعم الواجهة، حوض مرجاني خاص) — كل الحقول معبأة بمحتوى عربي احترافي واقعي.
- **3 مشاريع بهيكل جاهز** بحقول أقل تفصيلاً، سهلة التعبئة لاحقاً عند وصول الصور والبيانات الحقيقية.
- كل مشروع له 2-3 صور مكررة من المكتبة الحالية حتى ترفع صورك الأصلية (سهل الاستبدال لاحقاً).

## 3. تطوير النافذة المنبثقة (Modal)

تعاد كتابة الـ modal في `src/routes/portfolio.tsx` لتحوي الأقسام التالية بترتيب فاخر:

1. **معرض صور (Slider)**: صورة رئيسية كبيرة + thumbnails جانبية، أسهم تنقل، عداد (1/5)، دعم لمس للموبايل، تنقل لوحة مفاتيح.
2. **رأس**: التصنيف + العنوان + الموقع + السنة + مدة التنفيذ (chips ذهبية).
3. **النبذة**: فقرة وصف.
4. **شبكة المواصفات التقنية**: 4 بطاقات (الأبعاد / السعة / نوع الزجاج / النظام) بأيقونات Lucide.
5. **المعدات**: بطاقات (الفلتر، الإضاءة، التدفئة، CO₂/Wave) — تظهر فقط الحقول الموجودة.
6. **محتويات الحوض**: قائمة badges للأسماك + قائمة للنباتات/المرجان + الديكور.
7. **النطاق السعري**: شريط بارز ذهبي يعرض "يبدأ من X ر.س — حتى Y ر.س" بخط كبير + ملاحظة صغيرة "السعر يشمل التنفيذ والتركيب".
8. **CTA مزدوج**:
   - زر ذهبي رئيسي: **اطلب عرض سعر مماثل** → واتساب مع رسالة محضّرة تذكر اسم المشروع تلقائياً.
   - زر ثانوي: **استشارة مجانية**.

## 4. مكونات جديدة

- `src/components/ProjectGallery.tsx` — معرض صور بسلايدر RTL مع thumbnails (بدون مكتبات إضافية، useState فقط).
- `src/components/SpecCard.tsx` — بطاقة موحدة (أيقونة + label + value) لإعادة الاستخدام.

## 5. تحسينات إضافية لصفحة الأعمال

- بطاقة الشبكة تعرض **شارة سعر صغيرة** أعلى الصورة ("من 12,000 ر.س").
- إضافة badge **"مميّز"** ذهبي على المشاريع الكاملة كنماذج.
- زر **«شاهد التفاصيل»** ظاهر على hover.

## 6. تنظيم وإرسال صورك (إرشاد للمستخدم بعد التنفيذ)

سأرفق في الرد رسالة موجزة بالخطوات:
- مجلد لكل مشروع، تسمية واضحة.
- 10 صور كحد أقصى لكل رسالة شات.
- مع كل دفعة: اسم المشروع + الموقع + الأبعاد + النظام + الفلتر + الأسماك + النطاق السعري.
- بديل للكميات الكبيرة: رابط Google Drive.

---

## تفاصيل تقنية

- لا داعي لـ Lovable Cloud الآن — البيانات ثابتة في `src/data/projects.ts` (سهل التعديل، لا تحتاج لوحة تحكم).
- لا تغيير في الـ routes ولا الـ navbar.
- يبقى زر واتساب العائم والـ RTL كما هو.
- يتم تطبيق Reveal animations على الأقسام داخل الـ modal لتأثير fade-in.
- accessibility: focus trap داخل الـ modal، Escape للإغلاق، aria-labels على أزرار السلايدر.
